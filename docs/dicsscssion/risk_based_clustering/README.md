# Risk-Based Clustering Discussions

Working discussion notes for turning the outage trend / predictability review
layer into a more statistical county-pattern classifier.

These files are intentionally pre-methodology. They are for defining candidate
pattern shapes, deciding whether the labels are separable, and identifying the
statistical tests needed before any label becomes a pricing or underwriting
input.
