# Outputs

This folder is for generated curated-data reports and exports.

Generated outputs are local and reproducible. They should not be committed
unless explicitly promoted.

Examples:

```text
phase_1_noaa_match_report/
phase_1_oe417_match_report/
event_enriched_preview.csv
county_year_features_profile.html
```

The root `.gitignore` keeps generated output files out of git while preserving
this README.
